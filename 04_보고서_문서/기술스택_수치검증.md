## 5. 분석 성과 (수치 검증)

### 5.1 데이터 분석 결과
```
분석 대상: 24개 자치구
데이터 기준: CCTV(2024년), 범죄(2024년), 인구(2024년)

주요 통계:
  - 평균 CCTV 대수: 4,773대
  - 평균 범죄 발생: 3,252건
  - 평균 인구: 393,548명
  - 평균 CCTV효과범죄: 1,434건

상관분석 결과:
  - 전체 CCTV vs 전체 범죄 상관계수: 0.7684 (강한 양의 상관)
    p-value: 0.0000 (통계적 유의)

  - 방범 CCTV vs CCTV효과범죄 상관계수: 0.7408 (강한 양의 상관)
    p-value: 0.0000 (통계적 유의)
```

### 5.2 역인과관계 발견
- **상관계수**: 0.77 (강한 양의 상관관계)
- **해석**: CCTV 증가 → 범죄 감소(X), 범죄 다발지역 → CCTV 사후 설치(O)
- **정책적 시사점**: 24개 자치구를 4개 그룹으로 분류하여 우선순위 도출

### 5.3 4사분면 분류
- **Q1 (고CCTV/고범죄)**: CCTV 설치했으나 범죄 여전히 높음
- **Q2 (저CCTV/고범죄)**: CCTV 우선 설치 필요 지역 ⭐
- **Q3 (저CCTV/저범죄)**: 현 상태 유지
- **Q4 (고CCTV/저범죄)**: 효과적인 CCTV 운영 사례
