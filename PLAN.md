### Day 7 (7월 10일 수요일) - 회귀분석 (1) - 모델 구축

#### 목표
다중 선형 회귀 모델을 구축하여 CCTV와 범죄율의 관계를 정량화한다.

#### 세부 작업
- [ ] 회귀분석 설계
  - 종속변수: 인구당 범죄율 (CCTV 효과 범죄)
  - 독립변수 후보:
    - 인구당 방범CCTV
    - 인구밀도
    - 유동인구 (있을 경우)
    - 상권 지수 (있을 경우)
- [ ] 독립변수 간 다중공선성 확인
  - VIF (Variance Inflation Factor) 계산
  - VIF > 10인 변수 제거 또는 조정
- [ ] 단순 선형 회귀 (Baseline)
  - 종속변수: 인구당 범죄율
  - 독립변수: 인구당 방범CCTV만
  - 회귀계수, p-value, R² 확인
- [ ] 다중 선형 회귀 (Full Model)
  - 모든 독립변수 포함
  - Adjusted R² 확인
  - 각 독립변수의 유의성 확인 (p-value)
- [ ] 회귀 결과 해석
  - 방범CCTV 계수가 음수인가? (CCTV 많을수록 범죄율 낮음)
  - 통계적으로 유의미한가? (p < 0.05)
  - 설명력은? (Adjusted R² ≥ 0.3)

#### 산출물
- 회귀분석 결과 표
- 회귀계수 및 p-value 정리
- 회귀분석 노트북 (notebooks/07_regression_analysis.ipynb)

#### 목표

#### 세부 작업
- [ ] 회귀 가정 1: 선형성 (Linearity)

- [ ] 회귀 가정 3: 등분산성 (Homoscedasticity)
  - 잔차 vs 예측값 산점도
  - 패턴 존재 여부 확인
- [ ] 회귀 가정 4: 독립성 (Independence)
  - Durbin-Watson 통계량 (선택, 시계열 아니므로 참고용)
- [ ] 이상치 탐지
  - Cook's Distance 계산
  - 영향력 큰 관측치 식별
- [ ] 가정 위반 시 대응
  - 로그 변환 고려
  - 이상치 제거 또는 조정
  - Robust 회귀 (선택)
- [ ] 회귀분석 한계점 정리
  - 역인과성 문제 (범죄 많은 곳에 CCTV 추가 설치)
  - 단면 데이터 한계 (시간적 선후 관계 불명확)
  - 누락 변수 편향 (조명, 경찰 순찰 등 미포함)

#### 산출물
- 회귀 진단 그래프 (Q-Q plot, 잔차 plot 등)

### Day 9 (7월 12일 금요일) - 지역 분류 및 클러스터링

#### 목표
CCTV 밀도와 범죄율 기준으로 자치구를 4개 유형으로 분류한다.

#### 세부 작업
- [ ] 4사분면 분류 기준 설정
  - X축: 인구당 방범CCTV (중앙값 또는 평균 기준)
  - Y축: 인구당 범죄율 (중앙값 또는 평균 기준)
- [ ] 사분면별 자치구 분류
  - **1사분면 (고CCTV, 고범죄)**: "고위험/고커버리지 지역"
    - CCTV만으로 부족, 추가 대안 필요
  - **2사분면 (저CCTV, 고범죄)**: "고위험/저커버리지 지역"
    - CCTV 추가 설치 우선 지역
  - **3사분면 (저CCTV, 저범죄)**: "저위험/저커버리지 지역"
    - 현상 유지, 불필요 확충 지양
  - **4사분면 (고CCTV, 저범죄)**: "저위험/고커버리지 지역"
    - CCTV 효과 양호 또는 원래 안전한 지역
- [ ] 사분면 산점도 시각화
  - 각 사분면에 자치구명 표시
  - 중앙값 또는 평균선 표시
- [ ] 각 사분면 대표 자치구 선정 및 특성 분석
