## ❌ 부족한 데이터

### 1. 범죄 통계 데이터 (필수)

**필요 데이터**:
- 서울시 자치구별 범죄 발생 건수
- 범죄 유형별 분류 (절도, 강도, 차량범죄, 폭력, 성범죄 등)
- 연도: 2024년 또는 2025년

**추천 출처**:
1. **경찰청 범죄통계** (https://www.police.go.kr)
   - "경찰통계연보" 다운로드
   - 또는 공공데이터포털에서 "범죄 발생 통계" 검색

2. **공공데이터포털** (https://www.data.go.kr)
   - 검색어: "서울시 범죄 발생", "자치구별 범죄 통계"
   - 파일 형식: CSV 또는 Excel

3. **서울 열린데이터광장** (https://data.seoul.go.kr)
   - 검색어: "범죄", "치안", "5대 범죄"

**필요한 최소 필드**:
```
자치구명, 절도, 강도, 차량범죄, (선택: 폭력, 성범죄)
```

**현재 상태**: ❌ **미확보 - 즉시 다운로드 필요**

---

### 2. 인구 통계 데이터 (권장)

**필요 데이터**:
- 서울시 자치구별 인구수
- 면적 (km²)
- 연도: 2024년 또는 2025년

**추천 출처**:
1. **KOSIS 국가통계포털** (https://kosis.kr)
   - 국내통계 → 인구·가구 → 주민등록인구현황
   - 서울시 자치구별 인구 선택

2. **서울 열린데이터광장**
   - 검색어: "인구", "자치구별 인구"

**필요한 최소 필드**:
```
자치구명, 인구수, 면적_km2
```

**현재 상태**: ❌ **미확보 - 다운로드 권장**

**대안 (인구 데이터 없을 시)**:
- 인구 보정 없이 CCTV 절대값과 범죄 절대값만 분석
- 하지만 인구 차이가 큰 자치구 간 비교가 부정확해짐
- **강력 권장: 인구 데이터 확보**

---

### 3. 행정구 경계 데이터 (GeoJSON) (선택 - Day 6용)

**필요 데이터**:
- 서울시 25개 자치구 경계 폴리곤
- 파일 형식: GeoJSON 또는 SHP

**추천 출처**:
1. **공간정보 오픈플랫폼** (http://data.nsdi.go.kr)
2. **GitHub 공개 데이터**
   - 검색어: "seoul district geojson", "서울시 자치구 경계"
   - 예시: https://github.com/southkorea/seoul-maps

**현재 상태**: ⚠️ **미확보 - Day 6 전까지 다운로드**

**대안 (GeoJSON 없을 시)**:
- Choropleth 지도 시각화 생략
- 막대그래프 및 산점도로만 분석 진행 가능
