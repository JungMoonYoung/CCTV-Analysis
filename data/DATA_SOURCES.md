- **인코딩**: UTF-8 (또는 CP949)
- **다운로드 URL**: [실제 데이터 다운로드 시 기록]

#### 데이터 구조

### 2. 범죄 통계 데이터

#### 데이터 출처
- **제공 기관**: 경찰청 / 공공데이터포털
- **플랫폼**: 공공데이터포털 (https://www.data.go.kr)
- **데이터셋명**: 서울시 자치구별 범죄 발생 현황 (2023년)
- **파일 형식**: CSV / Excel
- **인코딩**: UTF-8 (또는 CP949)
- **다운로드 URL**: [실제 데이터 다운로드 시 기록]

#### 데이터 구조
- **행 개수**: 25개 (서울시 25개 자치구)
- **열 개수**: 7개
- **주요 필드**:
  - `자치구`: 자치구명
  - `절도`: 절도 범죄 건수
  - `강도`: 강도 범죄 건수
  - `차량범죄`: 차량 관련 범죄 건수
  - `공공장소폭력`: 공공장소 폭력 범죄 건수
  - `성범죄`: 성범죄 건수
  - `CCTV효과범죄_합계`: 절도+강도+차량범죄 합계
  - `총_범죄`: 전체 범죄 건수 합계

#### 데이터 품질
- **결측치**: 없음
- **이상치**: 일부 자치구에서 범죄 건수가 평균보다 높음 (유동인구 많은 지역)
- **데이터 연도**: 2023년 기준

#### CCTV 효과 범죄 정의
본 프로젝트에서는 CCTV가 억제 효과를 가질 것으로 예상되는 범죄만 선별:
- ✅ **절도**: 영상 감시 효과 기대
- ✅ **강도**: 대면형 범죄, 범행 전 심리 위축
- ✅ **차량범죄**: 주/공용구역 감시 효과
- ⚠️ **공공장소폭력**: 부분 포함 (해석 시 주의)
- ⚠️ **성범죄**: 별도 분리 분석 (민감 범죄)

#### 샘플 데이터 생성 방법
```python
np.random.seed(42)
crime_data = {
    '자치구': districts,
    '절도': np.random.randint(100, 800, size=25),
    '강도': np.random.randint(5, 50, size=25),
    '차량범죄': np.random.randint(50, 300, size=25),
    '공공장소폭력': np.random.randint(80, 400, size=25),
    '성범죄': np.random.randint(30, 150, size=25)
}
```

## 데이터 통합 계획

### 공통 키 (Join Key)
- **기준 필드**: `자치구` (자치구명)
- **통합 방법**: Inner Join (25개 자치구 모두 일치)

### 자치구명 표기 통일
모든 데이터셋에서 자치구명 표기를 동일하게 맞춤:
- 예시: "종로구", "중구", "용산구", ...
- 공백, 특수문자 제거
- 대소문자 통일 (한글이므로 해당 없음)

### 데이터 통합 후 최종 구조
```
통합 데이터프레임:
- 자치구 (25개)
- CCTV 관련 (총_CCTV, 방범용, 교통단속용, 어린이안전용, 기타)
- 범죄 관련 (절도, 강도, 차량범죄, 공공장소폭력, 성범죄, CCTV효과범죄_합계, 총_범죄)
- 인구 관련 (인구수, 면적_km2, 인구밀도)
- 파생 변수 (인구당_CCTV, 인구당_방범CCTV, 인구당_범죄율, 절도율, 강도율 등)
```
