## 문제 1: 자치구명 불일치로 인한 데이터 병합 실패

### 📌 발생 상황

3개의 CSV 파일(CCTV, 범죄, 인구)을 `자치구` 컬럼 기준으로 병합하는 과정에서 문제 발생

```python
# 데이터 병합 시도
df_cctv = pd.read_csv('data/raw/cctv_seoul_2024.csv', encoding='utf-8-sig')
df_crime = pd.read_csv('data/raw/crime_seoul_2024.csv', encoding='utf-8-sig')
df_population = pd.read_csv('data/raw/population_seoul_2024.csv', encoding='utf-8-sig')

merged = df_cctv.merge(df_crime, on='자치구', how='inner')
merged = merged.merge(df_population, on='자치구', how='inner')

print(f"예상: 25개 자치구")
print(f"실제: {len(merged)}개 자치구")  # 결과: 18개
```

**문제**: 25개 자치구가 아닌 18개만 병합됨 (7개 누락)

### 🔍 원인 분석

#### 1단계: 각 파일의 자치구명 확인

```python
print("CCTV 파일:", df_cctv['자치구'].unique())
# ['종로구', '중구', '용산구', ...]

print("범죄 파일:", df_crime['자치구'].unique())
# ['종로', '중구', '용산', ...]  # '구' 없음!

print("인구 파일:", df_population['자치구'].unique())
# ['종로구', '중구', '용산구', ...]
```

**발견**: 범죄 파일만 '구'가 빠져 있음

#### 2단계: 불일치 패턴 분석

```python
# 병합되지 않은 자치구 확인
cctv_districts = set(df_cctv['자치구'])
crime_districts = set(df_crime['자치구'])

missing = cctv_districts - crime_districts
print(f"병합 실패: {missing}")
# 출력: {'종로구', '중구', '용산구', ...}
```

**원인 파악**:
- CCTV 파일: "종로**구**", "중**구**"
- 범죄 파일: "종로", "중"
- 인구 파일: "종로**구**", "중**구**"

→ 범죄 데이터만 '구' 접미사가 없어 매칭 실패

### 💡 해결 과정

#### 시도 1: 범죄 파일에 '구' 추가

```python
# 잘못된 방법 (중구 처리 오류)
df_crime['자치구'] = df_crime['자치구'] + '구'

print(df_crime['자치구'].unique())
# ['종로구', '중구구', '용산구', ...]  # 중구구??? 오류!
```

**문제**: "중" → "중구구"로 잘못 변환됨

#### 시도 2: 조건부 처리

```python
def standardize_district_name(name):
    """자치구명 표준화"""
    # 이미 '구'로 끝나면 그대로
    if name.endswith('구'):
        return name
    # '구'가 없으면 추가
    else:
        return name + '구'

df_crime['자치구'] = df_crime['자치구'].apply(standardize_district_name)

print(df_crime['자치구'].unique())
# ['종로구', '중구', '용산구', ...]  # 정상!
```

**결과**: 25개 자치구 중 24개 병합 성공 (종로구 1개 여전히 누락)

#### 시도 3: 종로구 특별 처리

```python
# 종로구가 여전히 누락된 이유 확인
print(df_crime[df_crime['자치구'].str.contains('종로')])
# 결과: 빈 DataFrame

print(df_crime['자치구'].unique())
# ['중구', '용산구', ...] → 종로구 데이터 자체가 없음!
```

**발견**: 범죄 데이터에 종로구가 처음부터 없었음 (데이터 수집 오류)

### ✅ 최종 해결 방안

**1. 자치구명 표준화 함수 작성**

```python
def standardize_district_name(name):
    """
    자치구명을 표준 형식(OO구)으로 변환

    Args:
        name (str): 원본 자치구명

    Returns:
        str: 표준화된 자치구명
    """
    name = str(name).strip()  # 공백 제거

    # 이미 '구'로 끝나면 그대로 반환
    if name.endswith('구'):
        return name

    # '구'가 없으면 추가
    return name + '구'

# 모든 데이터프레임에 적용
df_cctv['자치구'] = df_cctv['자치구'].apply(standardize_district_name)
df_crime['자치구'] = df_crime['자치구'].apply(standardize_district_name)
df_population['자치구'] = df_population['자치구'].apply(standardize_district_name)
```



## 문제 2: 데이터 병합 시 중복 컬럼 발생

### 📌 발생 상황

3개 파일 병합 후 컬럼명이 중복되는 문제 발생

```python
merged = df_cctv.merge(df_crime, on='자치구', how='inner')
merged = merged.merge(df_population, on='자치구', how='inner')

print(merged.columns)
# ['자치구', 'CCTV_총계_x', 'CCTV_총계_y', '총인구_x', '총인구_y', ...]
```

**문제**: `_x`, `_y` 접미사가 붙어 컬럼명이 불명확해짐

### 🔍 원인 분석

#### 1단계: 중복 컬럼 확인

```python
# 각 파일의 컬럼 확인
print("CCTV 파일:", df_cctv.columns.tolist())
# ['자치구', 'CCTV_총계', '방범용', ...]

print("범죄 파일:", df_crime.columns.tolist())
# ['자치구', '총범죄_발생', '살인_발생', ...]

print("인구 파일:", df_population.columns.tolist())
# ['자치구', '총인구', '세대수', ...]
```

**발견**: 각 파일에 '자치구' 외 공통 컬럼이 없는데 왜 `_x`, `_y`가 생성되는가?

#### 2단계: 원인 파악

```python
# 두 번째 병합 시 문제 발생
merged_1 = df_cctv.merge(df_crime, on='자치구', how='inner')
print(merged_1.columns)
# 정상: ['자치구', 'CCTV_총계', '방범용', '총범죄_발생', ...]

merged_2 = merged_1.merge(df_population, on='자치구', how='inner')
print(merged_2.columns)
# 문제: ['자치구', 'CCTV_총계', ..., '총인구_x', '총인구_y']
```

**원인 파악**:
- 범죄 파일에도 '총인구' 컬럼이 있었음
- 인구 파일과 컬럼명이 겹쳐서 pandas가 자동으로 `_x`, `_y` 추가

#### 3단계: 실제 데이터 확인

```python
# 범죄 파일 상세 확인
print(df_crime.columns.tolist())
# ['자치구', '총범죄_발생', '살인_발생', ..., '총인구']  # 숨어있던 컬럼!

# 값이 동일한지 확인
```

**결론**: 범죄 파일에 불필요한 '총인구' 컬럼이 포함되어 있었음

### 💡 해결 과정

#### 시도 1: 병합 후 컬럼 이름 변경

```python
# 잘못된 접근

merged = merged.merge(df_population, on='자치구', how='inner')

```

**문제**:
- 매번 수동으로 처리해야 함
- 다른 중복 컬럼이 있으면 놓칠 수 있음

#### 시도 2: 병합 전 불필요한 컬럼 제거

```python
# 범죄 파일에서 '총인구' 컬럼 제거

```

**문제**:
- 하드코딩 (다른 중복 컬럼이 있으면 수동 추가)
- 스크립트 재사용성 낮음

### ✅ 최종 해결 방안

**1. 자동화된 중복 컬럼 제거 함수**

```python
def remove_duplicate_columns(df1, df2, key_column):
    """
    두 데이터프레임 병합 전 중복 컬럼 자동 제거

    Args:
        df1 (DataFrame): 첫 번째 데이터프레임
        df2 (DataFrame): 두 번째 데이터프레임
        key_column (str): 병합 키 컬럼명 (예: '자치구')

    Returns:
        DataFrame: 중복 컬럼이 제거된 df2
    """
    # df1과 df2의 공통 컬럼 찾기 (키 컬럼 제외)
    common_cols = set(df1.columns) & set(df2.columns)
    common_cols.discard(key_column)  # 병합 키는 제외

    if common_cols:
        print(f"⚠️ 중복 컬럼 발견: {common_cols}")
        print(f"   → df2에서 제거합니다.")
        df2_clean = df2.drop(columns=list(common_cols))
        return df2_clean
    else:
        return df2

# 사용 예시
df_crime_clean = remove_duplicate_columns(df_cctv, df_crime, '자치구')
df_population_clean = remove_duplicate_columns(
    df_cctv.merge(df_crime_clean, on='자치구'),
    df_population,
    '자치구'
)

merged = df_cctv.merge(df_crime_clean, on='자치구', how='inner')
merged = merged.merge(df_population_clean, on='자치구', how='inner')
```

**출력:**
```
⚠️ 중복 컬럼 발견: {'총인구'}
   → df2에서 제거합니다.
```

**2. 검증 로직 추가**

```python
# 병합 후 '_x', '_y' 접미사가 있는지 확인
suspicious_cols = [col for col in merged.columns if col.endswith(('_x', '_y'))]

if suspicious_cols:
```

**3. 명시적 컬럼 선택 (최종 방법)**

```python
def merge_with_column_selection(df1, df2, on, suffixes=('_left', '_right')):
    """
    명시적 컬럼 선택을 통한 안전한 병합

    Args:
        df1, df2: 병합할 데이터프레임
        on: 병합 키
        suffixes: 중복 시 사용할 접미사

    Returns:
        DataFrame: 병합된 데이터프레임
    """
    # 병합 수행
    merged = df1.merge(df2, on=on, how='inner', suffixes=suffixes)

    # 중복 컬럼 자동 처리
    for col in merged.columns:
        if col.endswith('_left'):
            original_name = col.replace('_left', '')
            right_col = original_name + '_right'

            if right_col in merged.columns:
                # 값이 동일한지 확인
                if merged[col].equals(merged[right_col]):
                    print(f"ℹ️ '{original_name}': 값 동일, _left 버전 유지")
                    merged = merged.drop(columns=[right_col])
                    merged = merged.rename(columns={col: original_name})
                else:
                    print(f"⚠️ '{original_name}': 값 다름, 수동 확인 필요!")

    return merged

# 사용
merged = merge_with_column_selection(df_cctv, df_crime, on='자치구')
merged = merge_with_column_selection(merged, df_population, on='자치구')
```
